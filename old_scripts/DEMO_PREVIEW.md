## 🎯 四大演示模块

### 1️⃣ 身份去匿名化

**可视化内容:**
- 左右对比显示原始图和匿名图
- 高亮显示正在匹配的节点（红色）
- 标记候选匹配节点（黄色）
- 显示最佳匹配结果（绿色）

**演示方法:**
1. **贪心特征匹配** - 逐个节点贪心选择
2. **匈牙利算法** - 全局最优匹配
3. **图核方法** - 基于子图结构
4. **DeepWalk嵌入** - 图嵌入空间匹配

**关键信息显示:**
```
节点 42 的特征:
  度数: 12
  聚类系数: 0.455
  邻居数: 12
  
最佳匹配: 节点 67
  相似度: 87.3%
  排名: 1/61
  
结果: ✅ 匹配正确
```

### 2️⃣ 属性推断

**可视化内容:**
- 不同颜色表示不同标签
- 已知标签节点彩色显示
- 未知标签节点灰色显示
- 逐步展示推断过程

**演示方法:**
1. **邻居投票** - 收集邻居标签投票
2. **标签传播** - 迭代式标签扩散
3. **GraphSAGE** - 神经网络聚合

**关键信息显示:**
```
预测节点 23 的标签:
  
邻居投票结果:
  标签0: 2票
  标签1: 5票 ← 多数
  标签2: 1票
  
预测: 标签1
真实: 标签1
结果: ✅ 预测正确
```

### 3️⃣ 鲁棒性测试

**可视化内容:**
- 左侧显示完整图
- 右侧显示不同缺失率的图
- 动态切换缺失场景
- 对比边数变化

**演示场景:**
```
缺失率 0%:  保留 270 条边 ✓ 基准
缺失率 10%: 保留 243 条边
缺失率 20%: 保留 216 条边
缺失率 30%: 保留 189 条边 ← 当前
缺失率 40%: 保留 162 条边
缺失率 50%: 保留 135 条边
```

### 4️⃣ 防御机制

**可视化内容:**
- 左侧显示原始图
- 右侧显示添加噪声后的图
- 不同ε值对比
- 高亮新添加的边

**防御参数对比:**
```
ε = 0.5:  添加 27 条噪声边  [强隐私保护]
ε = 1.0:  添加 13 条噪声边  [中等保护] ← 当前
ε = 2.0:  添加 7 条噪声边   [弱保护]
ε = 5.0:  添加 3 条噪声边   [最小保护]
```

## 🎮 交互操作

### 鼠标操作

- **点击拖动节点** - 调整节点位置
- **滚轮缩放** - 放大/缩小视图
- **拖动空白** - 平移整个图
- **悬停节点** - 显示详细信息

### 控制按钮

- **▶️ 开始演示** - 自动播放所有步骤（每步2秒）
- **⏸️ 暂停** - 暂停自动播放
- **⏭️ 下一步** - 手动执行下一步
- **🔄 重置** - 回到初始状态

### 阶段切换

点击顶部的阶段按钮即可切换：
```
[🎯 阶段1: 身份去匿名化]  ← 活跃
[🏷️ 阶段2: 属性推断]
[🛡️ 阶段3: 鲁棒性测试]
[🔒 阶段4: 防御机制]
```

## 📝 技术细节

### 可视化技术栈

- **D3.js v7** - 数据驱动的可视化
- **Force-directed Layout** - 力导向图布局
- **CSS3 Animations** - 平滑动画效果
- **Responsive Design** - 响应式设计

### 数据流程

```
Python脚本
  ↓ 生成JSON数据
HTML文件
  ↓ D3.js解析
SVG图形
  ↓ 用户交互
动态更新
```

